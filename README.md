# ReuzeController
a repository for the renovation of the giant controller made in 2020-2021
Authors: Degeest Bram, De Smet Xander.
Renovation: 2026

## about
The giant controller is (like the name says) a giant controller. It is moddeld after the SNES controller from nintendo. 
It has 10 buttons and can be used via bluetooth. The buttons work with a IR sensor that detects button presses. Originaly the controller used a Nucleo board as it's microcontroller with a bluetooth module attacht. the buttons are made out of 3D printed parts.
